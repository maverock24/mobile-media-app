# Code Of Conduct

## Our Standard

This project expects respectful, constructive collaboration.

Examples of behavior that contribute to a healthy environment:

- giving actionable, technical feedback
- assuming good intent while still being precise about problems
- welcoming contributors with different levels of experience
- focusing discussion on code, behavior, and decisions rather than people

Examples of unacceptable behavior:

- harassment, discrimination, or abusive language
- personal attacks or hostile review behavior
- repeated bad-faith arguments that block progress
- sharing private information without permission

## Scope

This code of conduct applies to repository discussions, issues, pull requests, review comments, and other project spaces.

## Enforcement

Maintainers may edit, remove, or reject contributions and discussions that do not meet these standards.

## Reporting

If you need to report unacceptable behavior, contact the project maintainers through the repository's primary communication channel and include enough context for the issue to be reviewed.
